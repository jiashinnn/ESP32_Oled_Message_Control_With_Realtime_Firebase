IoT Credential & Data Display System
A modern web app for Assignment 2 (STTHK3113).
Remotely manage your ESP32, WiFi credentials, and display messages in real time using Firebase and OLED.
Why Use This System?

Live Device Monitoring

Instantly see your ESP32's status and messages on the OLED display, updated in real time from anywhere.

Remote Configuration

Change device settings, update messages, and manage credentials without physical access to the device.

Secure Access

Only authorized users can log in and control your IoT devices, thanks to Firebase Authentication.

How It Works

Register

Create your account to access the dashboard and device controls.

Connect Device

Flash the ESP32, connect to WiFi, and link to Firebase for seamless data sync.

Control & Monitor

Use the web app to update messages and monitor device status in real time.

Use Case Example: Smart Noticeboard System
Imagine a university or office with a digital noticeboard powered by ESP32 and an OLED display. Staff can log in to this web app from anywhere, update the announcement or schedule in Firebase, and the new message appears instantly on the noticeboard. No need for physical access or manual updates—everything is managed remotely, securely, and in real time.

This system enables:
  • Centralized, remote control of public or private display boards
  • Instant updates for announcements, events, or alerts
  • Secure access—only authorized staff can change the message
